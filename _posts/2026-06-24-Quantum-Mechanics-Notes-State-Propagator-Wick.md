## 0. 总体问题：单粒子量子力学究竟在解决什么？

单粒子量子力学的核心任务是：

\[
\boxed{
\text{给定一个粒子所处的环境，预测它未来被测量到各种物理量的概率。}
}
\]

在经典力学中，关心的是粒子的确定轨道：

\[
x(t),\quad p(t)
\]

而在量子力学中，我再认为微观粒子一定有一条确定轨道，而是用量子态描述它。

单粒子量子力学主要回答三个问题：

1. 粒子可以处于什么状态？
2. 这些状态如何随时间演化？
3. 测量位置、动量、能量等物理量时，会得到什么结果以及对应概率是多少？

因此，量子力学不是在求"粒子到底走哪条路"，而是在求：

\[
\boxed{
\text{量子态如何存在、如何演化、以及测量结果的概率分布。}
}
\]

---

## 1. 量子态与波函数

### 1.1 量子态是什么？

量子态是希尔伯特空间中的抽象向量：

\[
|\psi\rangle \in \mathcal H
\]

它不是普通三维空间中的向量，而是一个抽象的数学对象。

如果一个粒子在三维空间中运动，那么：

\[
\mathbf r \in \mathbb R^3
\]

是粒子的位置变量所在的空间。

但是波函数所属的空间是：

\[
\psi \in L^2(\mathbb R^3)
\]

也就是说：

\[
\boxed{
\mathbb R^3 \text{ 是构型空间，} L^2(\mathbb R^3) \text{ 是波函数所在的希尔伯特空间。}
}
\]

---

### 1.2 波函数是什么？

在位置表象下，量子态可以写成波函数：

\[
\psi(x)=\langle x|\psi\rangle
\]

这表示抽象量子态 $\vert\psi\rangle$ 在位置基底 $\vert x\rangle$ 下的坐标表示。

所以更准确地说：

\[
\boxed{
\text{波函数不是量子态本身，而是量子态在某个表象下的表达。}
}
\]

在位置表象中：

\[
\psi(x)
\]

是位置空间中的概率幅。

真正的概率密度不是 $\psi(x)$，而是：

\[
|\psi(x)|^2
\]

所以：

\[
\boxed{
|\psi(x)|^2 = \text{粒子在位置 }x\text{ 附近被发现的概率密度。}
}
\]

---

### 1.3 波函数为什么不是普通概率？

因为波函数一般是复数：

\[
\psi(x)=A(x)e^{i\theta(x)}
\]

它既有振幅 $A(x)$，也有相位 $\theta(x)$。

概率密度只来自模平方：

\[
|\psi(x)|^2=A^2(x)
\]

但是相位也有物理意义。

比如：

\[
\psi_1(x)=A(x)
\]

和

\[
\psi_2(x)=A(x)e^{ikx}
\]

它们的位置概率密度相同：

\[
|\psi_1(x)|^2=|\psi_2(x)|^2
\]

但是它们的动量不同。

因此，量子态不是普通概率分布，而是：

\[
\boxed{
\text{带有相位信息的概率幅。}
}
\]

相位决定了干涉、动量分布以及后续演化。

---

### 1.4 归一化条件

对于普通可归一化的单粒子波函数，总概率应该等于 1：

一维中：

\[
\int_{-\infty}^{+\infty}|\psi(x,t)|^2dx=1
\]

三维中：

\[
\int_{\mathbb R^3}|\psi(\mathbf r,t)|^2d^3r=1
\]

意思是：

\[
\boxed{
\text{粒子总会在某个地方被发现，总概率为 1。}
}
\]

但是严格的平面波：

\[
\psi(x)=Ae^{ikx}
\]

在无限空间中不能普通归一化，因为：

\[
\int_{-\infty}^{+\infty}|Ae^{ikx}|^2dx=\infty
\]

所以平面波是理想化状态，真实粒子一般更接近局域波包。

---

## 2. 为什么有位置表象和动量表象？

### 2.1 表象就是选择基底

量子态本身是抽象的：

\[
|\psi\rangle
\]

但我们可以选择不同的基底去展开它。

位置表象使用位置本征态 $\vert x\rangle$ 作为基底：

\[
\psi(x)=\langle x|\psi\rangle
\]

动量表象使用动量本征态 $\vert p\rangle$ 作为基底：

\[
\phi(p)=\langle p|\psi\rangle
\]

所以：

\[
\boxed{
\text{表象就是选择一组基底来表示同一个量子态。}
}
\]

---

### 2.2 位置表象的意义

位置表象下：

\[
\psi(x)=\langle x|\psi\rangle
\]

它告诉我们：

\[
\boxed{
\text{这个量子态在不同位置上的概率幅是多少。}
}
\]

于是：

\[
|\psi(x)|^2
\]

是位置概率密度。

---

### 2.3 动量表象的意义

动量表象下：

\[
\phi(p)=\langle p|\psi\rangle
\]

为：

\[
\boxed{
\text{这个量子态包含多少动量为 }p\text{ 的成分。}
}
\]

于是：

\[
|\phi(p)|^2
\]

就是动量概率密度。

---

### 2.4 位置表象和动量表象的关系

位置表象和动量表象之间由傅里叶变换联系：

\[
\phi(p)
=

\frac{1}{\sqrt{2\pi\hbar}}
\int_{-\infty}^{+\infty}
e^{-ipx/\hbar}\psi(x)\,dx
\]

反过来：

\[
\psi(x)
=

\frac{1}{\sqrt{2\pi\hbar}}
\int_{-\infty}^{+\infty}
e^{ipx/\hbar}\phi(p)\,dp
\]

这说明：

\[
\boxed{
\text{动量表象就是把位置波函数分解成不同动量的平面波成分。}
}
\]

---

## 3. 动量本征态和平面波

### 3.1 动量算符

在位置表象中，动量算符是：

\[
\hat p=-i\hbar\frac{d}{dx}
\]

动量本征态满足：

\[
\hat p\psi_p(x)=p\psi_p(x)
\]

代入动量算符：

\[
-i\hbar\frac{d}{dx}\psi_p(x)=p\psi_p(x)
\]

解得：

\[
\psi_p(x)=Ae^{ipx/\hbar}
\]

令：

\[
k=\frac{p}{\hbar}
\]

则：

\[
\psi_p(x)=Ae^{ikx}
\]

这就是平面波。

因此：

\[
\boxed{
\text{动量本征态在位置表象下就是平面波。}
}
\]

---

### 3.2 $p=\hbar k$ 是否说明动量量子化？

不一定。

\[
p=\hbar k
\]

只说明动量 $p$ 与波数 $k$ 成正比。

动量是否离散，取决于 $k$ 是否离散。

如果粒子在无限空间中运动：

\[
k\in \mathbb R
\]

那么：

\[
p=\hbar k
\]

也是连续的。

如果粒子被限制在长度为 $L$ 的盒子中，边界条件会限制允许的波长：

\[
k_n=\frac{n\pi}{L}
\]

于是：

\[
p_n=\hbar k_n
\]

才出现离散动量。

所以：

\[
\boxed{
p=\hbar k \text{ 本身不代表动量量子化，边界条件才常常导致离散化。}
}
\]

---

### 3.3 平面波是谁的波？

量子力学中的平面波不是水波、声波，也不是电磁波。

它是：

\[
\boxed{
\text{粒子量子态在位置空间中的概率幅波。}
}
\]

一维平面波：

\[
\psi(x,t)=Ae^{i(kx-\omega t)}
\]

它的概率密度是：

\[
|\psi(x,t)|^2=|A|^2
\]

这说明粒子在各处出现的概率密度一样。

所以平面波代表：

\[
\boxed{
\text{动量完全确定，位置完全不确定的理想状态。}
}
\]

真实粒子通常不是严格平面波，而是波包：

\[
\psi(x)=\int A(k)e^{ikx}dk
\]

也就是说：

\[
\boxed{
\text{真实局域粒子可以看成许多不同动量平面波的叠加。}
}
\]

---

## 4. 量子化是干什么的？

量子化的核心作用是：

\[
\boxed{
\text{把经典物理系统改写成量子理论，从而描述微观世界。}
}
\]

经典力学中，一个粒子由位置和动量描述：

\[
x,\quad p
\]

经典哈密顿量是：

\[
H(x,p)=\frac{p^2}{2m}+V(x)
\]

量子化后：

\[
x\rightarrow \hat x
\]

\[
p\rightarrow \hat p
\]

\[
H(x,p)\rightarrow \hat H(\hat x,\hat p)
\]

在位置表象中：

\[
\hat x=x
\]

\[
\hat p=-i\hbar\frac{d}{dx}
\]

所以：

\[
\hat H
=

-\frac{\hbar^2}{2m}\frac{d^2}{dx^2}+V(x)
\]

于是薛定谔方程变成：
\[
i\hbar\frac{\partial}{\partial t}\psi(x,t)=
\hat H\psi(x,t)
\]

就是：

\[
i\hbar\frac{\partial}{\partial t}\psi(x,t)
=
\left[
-\frac{\hbar^2}{2m}\frac{d^2}{dx^2}+V(x)
\right]\psi(x,t)
\]

---

## 5. 量子化基于什么规则？

### 5.1 变量变算符

经典变量：

\[
x,\ p
\]

变成算符：

\[
\hat x,\ \hat p
\]

经典哈密顿量：

\[
H(x,p)
\]

变成量子哈密顿算符：

\[
\hat H(\hat x,\hat p)
\]

---

### 5.2 泊松括号变对易子

经典力学中有泊松括号：

\[
\{x,p\}=1
\]

正则量子化要求：

\[
\{A,B\}
\rightarrow
\frac{1}{i\hbar}[\hat A,\hat B]
\]

因此：

\[
[\hat x,\hat p]=i\hbar
\]

这条关系非常核心。

它说明位置和动量算符不能交换顺序，因此位置和动量不能同时任意精确确定。

由此可以得到不确定性关系：

\[
\Delta x\Delta p\geq \frac{\hbar}{2}
\]

---

### 5.3 对应原理

量子理论必须在宏观极限下回到经典理论。

也就是说：

\[
\hbar\to 0
\]

时，量子理论应该恢复经典力学。

这叫对应原理。

因此，量子化不是随意构造，而是要求：

\[
\boxed{
\text{微观尺度表现为量子行为，宏观极限恢复经典力学。}
}
\]

---

## 6. 传播子是干什么的？

传播子回答的问题是：

\[
\boxed{
\text{粒子从一个时空点传播到另一个时空点的概率幅是多少？}
}
\]

设初始波函数为：

\[
\psi(x',t')
\]

那么后来时刻 $t$ 的波函数可以写成：

\[
\psi(x,t)=\int K(x,t;x',t')\psi(x',t')dx'
\]

其中：

\[
K(x,t;x',t')
\]

就是传播子。

它表示：

\[
\boxed{
\text{粒子从 }(x',t')\text{ 到 }(x,t)\text{ 的概率幅。}
}
\]

传播子本质上是时间演化算符在位置表象下的矩阵元：

\[
K(x,t;x',t')=
\langle x|e^{-i\hat H(t-t')/\hbar}|x'\rangle
\]

---

## 7. 传播子和格林函数

传播子非常像格林函数，而且可以说：

\[
\boxed{
\text{传播子就是薛定谔方程的格林函数。}
}
\]

普通格林函数满足：

\[
LG(x,x')=\delta(x-x')
\]

量子传播子满足：

\[
\left(
i\hbar\frac{\partial}{\partial t}
-\hat H
\right)
K(x,t;x',t')
=

i\hbar\delta(t-t')\delta(x-x')
\]

所以它是算符：

\[
i\hbar\partial_t-\hat H
\]

的格林函数。

直观理解：

\[
\boxed{
\text{格林函数是点源响应，传播子是点源初态的量子演化响应。}
}
\]

---

## 8. 路径积分是干什么的？

路径积分给传播子一个新的表达方式：

\[
K(x_b,t_b;x_a,t_a)
=

\int_{x(t_a)=x_a}^{x(t_b)=x_b}
\mathcal D x(t)\,
e^{iS[x]/\hbar}
\]

这里：

\[
S[x]=\int_{t_a}^{t_b}L(x,\dot x,t)dt
\]

是路径 $x(t)$ 的作用量。

这句话的意思是：

\[
\boxed{
\text{粒子从 A 到 B 的总概率幅，是所有可能路径贡献的相位叠加。}
}
\]

每一条路径贡献：

\[
e^{iS[x]/\hbar}
\]

经典路径不是被手动指定的，而是在 $\hbar\to 0$ 时由驻相近似自然出现：

\[
\delta S=0
\]

也就是经典最小作用量原理。

因此路径积分连接了：

1. 量子干涉；
2. 经典最小作用量；
3. 传播子；
4. 量子场论；
5. 统计物理；
6. Wick 转动。

---

## 9. Wick 转动是干什么的？

Wick 转动是把真实时间变成虚时间：

\[
t=-i\tau
\]

实时间演化因子：

\[
e^{-iEt/\hbar}
\]

变成：

\[
e^{-E\tau/\hbar}
\]

也就是说：

\[
\boxed{
\text{实时间中的相位振荡，变成虚时间中的指数衰减。}
}
\]

这不是说真实世界中粒子真的沿着虚时间衰减，而是说：

\[
\boxed{
\text{虚时间是一种能保留物理能谱信息的计算表象。}
}
\]

实时间中，能量表现为振荡频率：

\[
e^{-iEt/\hbar}
\]

虚时间中，能量表现为衰减率：

\[
e^{-E\tau/\hbar}
\]

因此，通过虚时间相关函数的衰减率，可以读出真实系统的能量差。

例如：

\[
C_E(\tau)\sim e^{-\Delta E\tau/\hbar}
\]

则：

\[
\Delta E
\]

就是真实系统的能级差。

所以 Wick 转动的物理意义是：

\[
\boxed{
\text{把实时间的振荡问题，变成虚时间的衰减问题，从而更方便地提取能谱和基态信息。}
}
\]

---

## 10. Wick 转动与热核

实时间自由粒子传播子形式类似：

\[
K(x,t;x',0)
=

\sqrt{\frac{m}{2\pi i\hbar t}}
\exp\left[
\frac{im(x-x')^2}{2\hbar t}
\right]
\]

这是一个振荡核。

做 Wick 转动：

\[
t=-i\tau
\]

得到欧几里得传播子：

\[
K_E(x,\tau;x',0)
=

\sqrt{\frac{m}{2\pi\hbar\tau}}
\exp\left[
-\frac{m(x-x')^2}{2\hbar\tau}
\right]
\]

这是高斯衰减核，很像热方程的格林函数。

因此：

\[
\boxed{
\text{实时间传播子像波动核，虚时间传播子像热核。}
}
\]

这也是 Wick 转动和统计物理之间产生深刻联系的原因。

## 11. 从微分几何角度类比

可以把量子态和表象类比为微分几何中的抽象向量与坐标表示。

在微分几何中，一个切向量 $v\in T_pM$ 是抽象对象。

选定坐标后，它可以写成：

\[
v=v^i\frac{\partial}{\partial x^i}
\]

这里 $v^i$ 是坐标分量。

换一组坐标，分量会变，但向量本身不变。

量子力学类似：

\[
|\psi\rangle
\]

是抽象量子态。

位置表象：

\[
\psi(x)=\langle x|\psi\rangle
\]

动量表象：

\[
\phi(p)=\langle p|\psi\rangle
\]

同一个量子态在不同表象下有不同表达，但态本身不变。

对应关系可以整理为：

| 微分几何 | 量子力学 |
| ---------- | ----------------------- |
| 抽象向量 $v$ | 抽象量子态 $\vert\psi\rangle$ |
| 坐标系/标架 | 表象/基底 |
| 坐标分量 $v^i$ | 波函数 $\psi(x)$、$\phi(p)$ |
| 换坐标 | 换表象 |
| 向量不变，分量变 | 态不变，波函数表达变 |

因此更准确的说法是：

\[
\boxed{
\text{量子态是希尔伯特空间中的抽象向量，波函数是它在某个表象下的坐标表达。}
}
\]

## 12. 总结：这条学习线索如何串起来？

这几个概念之间的逻辑关系可以整理为：

### 第一层：量子态

\[
|\psi\rangle
\]

量子态是系统的完整描述。

---

### 第二层：表象

位置表象：

\[
\psi(x)=\langle x|\psi\rangle
\]

动量表象：

\[
\phi(p)=\langle p|\psi\rangle
\]

表象是量子态的不同坐标表示。

---

### 第三层：算符

物理量由算符表示：

\[
\hat x,\quad \hat p,\quad \hat H
\]

测量结果是算符的本征值。

---

### 第四层：量子化

经典系统：

\[
H(x,p)
\]

量子化为：

\[
\hat H(\hat x,\hat p)
\]

并满足：
\[
[\hat x,\hat p]=i\hbar
\]

---

### 第五层：薛定谔方程

时间演化由：

\[
i\hbar\frac{\partial}{\partial t}|\psi(t)\rangle
=

\hat H|\psi(t)\rangle
\]

决定。

---

### 第六层：传播子

传播子是时间演化算符的核：

\[
K(x,t;x',t')
=

\langle x|e^{-i\hat H(t-t')/\hbar}|x'\rangle
\]

它也是薛定谔方程的格林函数。

---

### 第七层：路径积分

传播子可以写成所有路径的相位叠加：

\[
K=
\int\mathcal D x\,
e^{iS[x]/\hbar}
\]

---

### 第八层：Wick 转动

把：

\[
t=-i\tau
\]

后：

\[
e^{iS/\hbar}
\rightarrow
e^{-S_E/\hbar}
\]

振荡问题变成衰减问题，从而可以提取基态、能谱和统计物理信息。

---

## 13. 最终一句话

单粒子量子力学的主线可以概括为：

\[
\boxed{
\text{用希尔伯特空间中的量子态描述粒子，用表象表示量子态，用算符表示物理量，用薛定谔方程描述演化，用传播子和路径积分表达传播幅。}
}
\]

而 Wick 转动进一步告诉我们：

\[
\boxed{
\text{实时间中的量子相位振荡，可以在虚时间中转化为指数衰减，从而更方便地提取真实物理信息。}
}
\]
